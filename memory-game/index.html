Instructions

Matchy Matchy, Match Match, Concentration, Memory, Pelmanism, Shinkei-suijaku -- the game has many names. However, the game's objective stays the same: turn over pairs of matching cards based on memory (or luck!).

NE67: 4,000+ foot peaks in New England

This game matches New England mountain peaks instead of the typical playing cards. You'll test your memory, and learn about a few beautiful places in New England.
